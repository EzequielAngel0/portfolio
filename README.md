## 🌟 Sobre el Proyecto

Este es mi portafolio profesional, diseñado y desarrollado desde cero para destacar mi trayectoria, proyectos, certificaciones y habilidades en el desarrollo de software. Construido con un enfoque en el rendimiento, accesibilidad y diseño moderno (Premium UI con *Glassmorphism* y animaciones fluidas).

### ✨ Características Principales

- 📱 **Diseño Responsivo y Moderno:** UI/UX cuidadosamente elaborada con Tailwind CSS, temática oscura (Dark Mode), desenfoques y gradientes interactivos.
- ⚡ **Rendimiento Extremo (Astro):** Aprovecha la arquitectura de *Zero JS by default* de Astro para tiempos de carga ultrarrápidos.
- 🧩 **Arquitectura Modular:** Componentes reutilizables diseñados bajo principios limpios.
- 🔄 **Despliegue Continuo (CI/CD):** Configurado con **GitHub Actions** para compilar y publicar automáticamente en GitHub Pages ante cada cambio en el código.

## 🛠️ Tech Stack & Herramientas

### Desarrollo del Portafolio
- **Framework:** [Astro](https://astro.build/) - Framework web estático de altísimo rendimiento.

- **Lenguaje:** [TypeScript](https://www.typescriptlang.org/) - Tipado estático y robusto.
- **Hosting & CI/CD:** GitHub Pages + GitHub Actions.

Como Ingeniero de Software, me especializo en construir soluciones robustas integrando código desde el servidor hasta interfaces dinámicas:

## 🚀 Instalación y Desarrollo Local

Si deseas explorar o compilar este proyecto de manera local, sigue estos pasos:

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/EzequielAngel0/portfolio.git
   cd portafolio
   ```

2. **Instalar dependencias:**
   ```bash
   npm install
   ```

3. **Iniciar servidor de desarrollo:**
   ```bash
   npm run dev
   ```
   *El proyecto estará disponible en `http://localhost:4321`*

4. **Compilar para producción:**
   ```bash
   npm run build
   ```
